01.db5free免費資料庫

db4free 線上免費資料庫

工具 說明
host db4free.net
mysql 8.0.16
port 3306

注意

  1. 使用此線上資料庫 無時間限制
  2. 容量大小 200 MB
  3. MySql 版本更新紀錄 可查詢，並使用兼容的 mysql 版本
  4. 僅供測試使用 (可能會發生停機、資料遺失，且無法提供一般專業數據供應商有的資訊安全保護措施)
  5. 線上連線慢

總結

比較
連線 慢
容量 200MB
版本 持續更新
資料 易遺失

連線若有問題，需至 Blog 查看，可參考文章方法。 但仍需自行測試是否連線成功(如 3306 port 連線失敗，就改 3307)
