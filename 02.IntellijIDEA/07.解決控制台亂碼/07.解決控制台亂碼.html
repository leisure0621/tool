07.解決控制台亂碼

設定 IDEA 編碼

  • File > Setting > Editor

設定 VM Options

  • Help > Edit Custom VM Options...

  • 添加參數 -Dconsole.encoding=UTF-8 、-Dfile.encoding=UTF-8
-Xms128m
-Xmx2027m
-XX:ReservedCodeCacheSize=512m
-XX:+UseG1GC
-XX:SoftRefLRUPolicyMSPerMB=50
-XX:CICompilerCount=2
-XX:+HeapDumpOnOutOfMemoryError
-XX:-OmitStackTraceInFastThrow
-ea
-Dsun.io.useCanonCaches=false
-Djdk.http.auth.tunneling.disabledSchemes=""
-Djdk.attach.allowAttachSelf=true
-Djdk.module.illegalAccess.silent=true
-Dkotlinx.coroutines.debug=off
-Dconsole.encoding=UTF-8
-Dfile.encoding=UTF-8

設定 Debug Configurations

  • Run > Debug Configurations > Server > VM Options: -Dfile.encoding=UTF-8

  • Startup/Connection > + > JAVA_TOOL_OPTIONS > -Dfile.encoding=UTF-8

注意

  • 上述動作完成後都需 重啟 IDEA
  • IDEA中Tomcat在控制台乱码问题以及IDEA编码设置UTF-8
  • 解决IntelliJ IDEA控制台输出中文乱码问题
  • 【IntelliJ IDEA】tomcat啟動，列印日誌亂碼問題
