02. NodeJs 安裝

使用 node.js官方下載地址，或本站 下載地址 進行 nodejs 的 v10.21.0 版本下載並安裝，用 v10.21.0 才不容易出現後續 gitbook 安裝問題 (其實 一直下一步 就可以安裝完成)

  • 確認 npm 版本， "npm -v"

03. GitBook-cli 安裝

  • 開啟 CMD 窗口，並執行指令安裝 Gitbook
npm install -g gitbook-cli

  • 安裝完成

  • 進入欲製作 gitbook 的目錄，執行 "gitbook init" 初始化 GitBook 項目

  • 執行 "gitbook -V"，確認 gitbook 版本

  • 執行 "gitbook serve" 預覽編輯好的 GitBook 項目

04. GitBook 套件設定與安裝

  • 目錄

  • 創建並設定 book.json
{
  "plugins": [
    "-search",
    "search-pro",
    "back-to-top-button",
    "code",
    "pageview-count",
    "popup"
  ],
  "pluginsConfig": {
    "search-pro": {
      "cutWordLib": "nodejieba",
      "defineWord": ["小需求", "基础建设"]
    }
  }
}
  • 執行 "gitbook install" 安裝已設定的套件
